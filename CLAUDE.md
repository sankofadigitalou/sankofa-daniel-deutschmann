# Daniel Deutschmann Website

## Overview
Single-page artist website for Daniel Deutschmann — a German-African artist persona within the Sankofa Ecosystem.

## Tech Stack
- Pure HTML + CSS + Vanilla JavaScript (no framework)
- Google Fonts: Playfair Display, Space Grotesk, JetBrains Mono
- Bilingual: German (default) / English

## Design System
- Dark luxury aesthetic with earth tones + gold (Sankofa) accents
- Colors: `--bg-deep: #0a0908`, `--gold: #d4af37`, `--kente-red: #c8102e`
- Scroll reveal animations, particle effects, parallax hero

## Structure
- `index.html` — Complete single-page website
- Sections: Hero, About, Identity/Tension, Quotes, Journey/Timeline, Themes, Music, Connect, Footer
